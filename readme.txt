== Description ==

Very basic plugin just create custom post type(CPT) for tools.

Create tools website such as calculator, generator, converter, etc. easily in WordPress.

This plugin is used for creating and marinating tool websites in WordPress.

This plugin is "OK" for use but its UX is not extremely good.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Now you get "Web Tools" menu in your admin view.
1. Enjoy the plugin :)

== Frequently Asked Questions ==

= Is this plugin able to create any kind of tools or web-application? =

Well, it depends on your need in this plugin you get a code input field where you can paste your code but sometimes it's not possible for you to pass all your code on one page. That why it's not recommended for multi-page web applications.
